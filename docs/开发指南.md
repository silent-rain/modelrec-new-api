## Docker

### 将当前用户添加到 docker 组

```sh
# 检查 docker 组是否存在
grep docker /etc/group

# 如果不存在，创建 docker 组
sudo groupadd docker

# 将当前用户添加到 docker 组
sudo usermod -aG docker $USER

# 验证是否生效：
docker ps


# 修改 socket 权限（临时）
sudo chmod 666 /var/run/docker.sock

# 重启 Docker 服务
sudo systemctl restart docker
```

### 配置 Docker 镜像加速器

```sh
sudo mkdir -p /etc/docker
sudo nano /etc/docker/daemon.json
```

```json
{
  "registry-mirrors": [
    "https://docker.1ms.run",
    "https://docker.m.daocloud.io",
    "https://docker.xuanyuan.me"
  ]
}
```

重启 Docker:

```sh
sudo systemctl daemon-reload
sudo systemctl restart docker
```

### 本地镜像编译

```sh
docker build -t new-api:local .
```

## PostgreSQL

```sql
-- 创建用户
CREATE USER newapi WITH PASSWORD 'pass';


-- 创建数据库
CREATE DATABASE "new-api" OWNER newapi;
```

配置 PostgreSQL 允许外部连接:

```sh
# 修改 pg_hba.conf
# sudo vim /etc/postgresql/18/main/pg_hba.conf
# sudo vim /var/lib/postgres/data/pg_hba.conf
sudo vim /www/server/pgsql/data/pg_hba.conf
# 在文件末尾添加:
host    all    all    172.17.0.0/16    md5

# 修改 postgresql.conf
# sudo vim /etc/postgresql/18/main/postgresql.conf
# sudo vim /var/lib/postgres/data/postgresql.conf
sudo vim /www/server/pgsql/data/postgresql.conf
# 修改: listen_addresses = '*'

# 重启
sudo systemctl restart postgresql
```

## Redis

配置 Redis 允许外部连接:

```sh
# 修改 redis.conf
# sudo vim /etc/redis/redis.conf
sudo vim /www/server/redis/redis.conf
# 修改: bind 0.0.0.0
# 建议设置密码: requirepass your_strong_password

# 重启
sudo systemctl restart redis
```

验证步骤:

```sh
# 1. 重新构建并启动容器
docker-compose down
docker-compose up -d --build

# 2. 查看日志确认数据库连接成功
docker logs new-api -f

# 3. 如果连接失败，测试网络连通性
docker exec -it new-api ping host.docker.internal

```
